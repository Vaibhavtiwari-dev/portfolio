## Overview

Personal portfolio site built as a single-page editorial design. Clean typography, structured layout, and inline styles — no framework, no build step.

## Tech Stack

- **HTML5** — semantic, accessible markup
- **CSS3** — inline styles with custom properties, fully responsive
- **JavaScript** — minimal inline JS (mobile menu only)
- **Firebase Hosting** — deployment with redirect rules and cache headers

## Project Structure

```
├── index.html       # Main portfolio (editorial single-page)
├── resume.html      # Print-ready resume page
├── resume.css       # Resume-specific styles
├── 404.html         # Custom 404 (editorial theme)
├── favicon.png      # Compressed favicon (64x64)
├── favicon.svg      # SVG favicon
├── profile.jpg      # Profile photo
├── robots.txt       # Search crawler rules
├── sitemap.xml      # Sitemap for SEO
├── firebase.json    # Hosting config + redirects
└── .firebaserc      # Firebase project reference
```

## Running Locally

```bash
git clone https://github.com/Vaibhavtiwari-dev/Vaibhav-Portfolio.git
cd Vaibhav-Portfolio
npx serve .
```
